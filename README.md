## 📝 Refleksi Mahasiswa
# 1. Apa perbedaan Bun.js dan Node.js menurut pemahaman Anda?

Bun.js merupakan JavaScript runtime modern yang memiliki performa lebih cepat dan lebih ringan dibandingkan Node.js. Bun.js juga menyediakan package manager bawaan serta mendukung TypeScript secara default, sehingga proses pengembangan menjadi lebih efisien dan praktis. Sementara itu, Node.js adalah JavaScript runtime yang telah lebih lama digunakan dan memiliki ekosistem library yang sangat besar serta komunitas yang luas. Hal ini membuat Node.js lebih stabil dan kaya referensi dalam pengembangan aplikasi berskala besar.

# 2. Mengapa Git penting dalam pengembangan perangkat lunak?

Git sangat penting karena memungkinkan developer untuk mengelola versi kode, melacak setiap perubahan, serta memudahkan kolaborasi dalam tim pengembang. Dengan adanya Git, setiap perubahan dapat terdokumentasi dengan baik sehingga meminimalkan risiko kehilangan data. Selain itu, Git membantu proses pengembangan menjadi lebih terstruktur, aman, dan efisien, terutama ketika proyek dikerjakan secara berkelompok maupun dalam jangka waktu yang panjang.
